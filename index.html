Vec3.add()

Using the Vec3.add() method we can add a and b together.


    let a = new Vec3(1, 0, 0);
    let b = new Vec3(0, 1, 0);

    a.add(b); // modifies a where the values in a are the summations in each axis
    console.log(a); // returns Vec3 {x: 1, y: 1, z: 0}

Vec3.subtract()

Using the Vec3.subtract() method we can subtract b from a.


    let a = new Vec3(1, 0, 0);
    let b = new Vec3(0, 1, 0);

    a.subtract(b); // modifies a where the values in a are the differences in each axis
    console.log(a); // returns Vec3 {x: 1, y: -1, z: 0}

Vec3.directionCosine() and Vec3.directionSine()

Vec3.directionCosine() finds the cosine of the angle between between the caller and the input vectors, and Vec3.directionSine() finds the sine of the angle between the vectors.


    let a = new Vec3(1, 0, 0);
    let b = new Vec3(0, 1, 0);

    a.directionCosine(b); // returns 0
    a.directionSine(b); // returns 1
